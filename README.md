Test repository for Codience demos.

## Jira Design Pack for Reviewer Recommendation

This repo now includes a Jira project design that models reviewer recommendation inputs from diverse PR risk and business impact values.

- `jira-design/project-blueprint.md` - Jira project structure, workflow, custom fields, and scoring design
- `jira-design/contributors.csv` - 5 contributors with domain strengths
- `jira-design/issues.csv` - 15 issues mapped to simulated PRs (`PR-201` to `PR-215`)
- `jira-design/jira-ticket-creation-guide.md` - step-by-step Jira website setup and CSV import guide

Use these files to create/import a Jira project and feed Codience recommendation signals.
